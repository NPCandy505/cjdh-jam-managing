# CJDH Jam Day

재즈 동아리의 잼 데이 곡 신청과 참여 현황 관리를 돕는 웹사이트 프로젝트입니다.

이 프로젝트의 목표는 다음과 같습니다.

1. 동아리원이 곡을 빠르게 신청할 수 있게 하기
2. 운영진이 신청 현황을 표 형태로 한눈에 확인할 수 있게 하기
3. 신청자 본인의 희망 세션과 추가 참여 멤버 정보를 명확하게 관리하기
4. 곡을 아직 못 정한 사람도 easy 곡 리스트와 원클릭 신청으로 쉽게 참여하게 하기

## 주요 기능

MVP 기준으로 다음 기능을 구현합니다.

1. 곡 신청 현황 표 조회
2. 신청곡 추가
3. 추가 참여 멤버 정보 관리
4. easy 곡 리스트 표시
5. 원클릭 신청 기능
6. 신청 수정/삭제

## 기술 스택

이 프로젝트는 아래 스택을 기준으로 개발합니다.

1. `Node.js`
   - 개발 서버 실행과 패키지 설치를 담당합니다
2. `Next.js`
   - 화면과 서버 기능을 한 프로젝트 안에서 구현합니다
3. `TypeScript`
   - 타입 기반으로 코드 안정성을 높입니다
4. `SQLite`
   - 신청곡, 원클릭 신청 등 데이터를 저장합니다
5. `Prisma`
   - 애플리케이션 코드와 데이터베이스를 연결합니다
6. `Vercel`
   - 추후 배포를 담당합니다

## 핵심 데이터 개념

1. `JamEntry`
   - 곡명, 신청자, 희망 세션, 추가 참여 멤버, 비고를 담는 신청 데이터
2. `AdditionalParticipant`
   - 신청자를 제외하고 해당 곡에 추가로 참여하는 멤버의 이름과 세션 정보
3. `EasySong`
   - easy 곡 추천 리스트 데이터
4. `OneClickRequest`
   - 아직 곡을 고르지 못한 사람의 원클릭 신청 데이터
